# Codex Development Instructions

- Read `SPEC.md` before modifying code.
- Use Vite, plain JavaScript modules, Firebase Anonymous Authentication and Realtime Database.
- Do not add a backend, Cloud Functions, Storage, Admin SDK, or open database rules.
- Keep student and teacher strokes in separate Canvas layers and database paths.
- Use Pointer Events, normalized coordinates and Retina Canvas rendering; optimize for iPad Safari.
- Draw locally before syncing, remove listeners when views change, and show errors in Traditional Chinese.
- Run `npm test` and `npm run build` after meaningful changes.
